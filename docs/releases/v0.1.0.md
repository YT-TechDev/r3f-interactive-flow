# r3f-interactive-flow v0.1.0

Initial release of `r3f-interactive-flow`.

`r3f-interactive-flow` is a small React Three Fiber utility library for building predictable phase-based interactive 3D experiences.

It connects:

- React-managed phase state
- transition progress
- `next` / `prev` / `goTo` controls
- wheel, touch, and keyboard input
- React Three Fiber frame updates

The goal is to provide a small control layer for interactive R3F websites without becoming a visual effects collection or a full animation framework.

## Highlights

- Added core phase machine support.
- Added `next`, `prev`, and `goTo` phase navigation.
- Added transition progress, transition direction, transition state, and lock state tracking.
- Added `FlowProvider`.
- Added `useFlow`.
- Added `useFlowProgress`.
- Added `useFlowFrame` for React Three Fiber frame-based updates.
- Added `useWheelInput`.
- Added `useTouchInput`.
- Added `useKeyboardInput`.
- Added a basic Vite example.
- Added TypeScript build output with ESM, CJS, and declaration files.
- Added Next.js App Router client-entry compatibility through the preserved `"use client"` directive.
- Added CI, tests, package dry-run, release readiness checks, and package output verification.
- Added npm-facing README, CHANGELOG, CONTRIBUTING guide, release checklist, PR template, and issue templates.

## Install

```bash
pnpm add r3f-interactive-flow three @react-three/fiber react react-dom
```

## Peer dependencies

This package expects the following peer dependencies to be installed by the app:

- `react`
- `react-dom`
- `three`
- `@react-three/fiber`

## Public API

The initial public API includes:

- `FlowProvider`
- `useFlow`
- `useFlowProgress`
- `useFlowFrame`
- `useWheelInput`
- `useTouchInput`
- `useKeyboardInput`
- `UseWheelInputOptions`
- `UseTouchInputOptions`
- `UseKeyboardInputOptions`

## Scope

This release intentionally keeps the project focused on predictable phase-based flow control.

It is not:

- a visual effects collection
- a replacement for `@react-three/drei`
- a particle library
- a camera preset library
- a shader effect library
- a portfolio template
- a full animation framework
- a GSAP or Framer Motion wrapper
