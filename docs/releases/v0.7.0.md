# r3f-interactive-flow v0.7.0

v0.7.0 is a narrow stabilization release focused on consumer package confidence after the v0.6.0 input stabilization pass.

This release improves package output verification, public API export confidence, README/example consistency, and release-prep documentation without expanding the runtime API.

## Improved

- Hardened package output verification for the current published package shape.
- Improved confidence that the documented public API matches the package entrypoint.
- Updated README and Vite basic example guidance for the current package behavior.
- Clarified release-prep documentation around verification, versioning, publishing, tags, and GitHub Releases.
- Added v0.7.0 roadmap documentation for the post-v0.6.0 stabilization pass.

## Scope

- No runtime dependency additions were made.
- No public runtime API expansion was introduced.
- No visual effects, camera presets, shader APIs, animation timelines, router integration, GSAP integration, or Framer Motion integration were added.
- No large examples or templates were added.
- No release automation behavior was changed.
- No npm publish, git tag creation, or GitHub Release creation is included in this release-prep PR.
